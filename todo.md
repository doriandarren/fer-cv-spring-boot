# Annotations

## Estuctura del proyecto para WEB:

```sh
/**
 * index -> lista
 * show -> ver uno
 * create -> cargar view para crear
 * store -> guardar
 * edit -> cargar view para editar
 * update -> actualizar
 * destroy -> eliminar
 */
```


## Estuctura del proyecto para API:

```sh
/**
 * index -> lista
 * show -> ver uno
 * store -> guardar
 * update -> actualizar
 * destroy -> eliminar
 */
```


## Comandos

```sh


./mvnw spring-boot:run                                      ## para ejecutar servidor hot-reload


./mvnw clean package                                        ## Compila el proyecto

java -jar target/tu-cv-spring-boot-0.0.1-SNAPSHOT.jar       ## Ejecutar el .jar

```
