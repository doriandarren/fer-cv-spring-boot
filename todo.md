# Annotations

## Estuctura del proyecto para WEB:

```sh
/**
 * findAll      -> index    -> lista
 * findById     -> show     -> ver uno
 * create                   -> cargar view para crear
 * save         -> store    -> guardar
 * edit                     -> cargar view para editar
 * save         -> update   -> actualizar
 * deleteById   -> destroy   -> eliminar
 */
```
